# RIVER MCU

RIVER MCU ("RISC-V Educational and Research Microcontroller
Unit") is a RISC-V compliant microprocessor system serving as a stand-alone primer on teach-
ing microprocessor design and programming.

It provides RV32I ISA, built-in UART, VGA terminal and easy to use debug interfaces.
More resources: https://github.com/AGH-engineering-thesis-SM-PK-2026
